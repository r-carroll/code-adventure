# コード・アドベンチャーへようこそ！ 🚀

こんにちは！これから、**Ruby**（プログラミング言語）と **Gemini CLI**（AIの相棒）を使って、自分だけのビデオゲームを作る冒険が始まります。

---

## 🐢 自分のペースで進めよう
プログラミングを学ぶのは、楽器の練習や新しいスポーツを覚えるのと同じで、時間がかかるものです。
- **焦らなくて大丈夫:** 1日で全部終わらせる必要はありません。
- **1日1レベル:** 1日に1レベルだけ、あるいは1週間に1レベルでも全く問題ありません。
- **楽しもう:** もし楽しくてどんどん進みたいなら、何レベル進めてもOKです！
一番大切なのは、次のステップに行く前に、そのコードが**なぜ**動いているのかをしっかり理解することです。

---

## 🛠 ステップ1：道具をそろえよう
プログラミングを始める前に、君のコンピューターを「武器屋」のようにセットアップしましょう。

### 0. 冒険を手に入れよう（コードをダウンロードする）
まずは、このプロジェクトのファイルを自分のコンピューターにコピーする必要があります！
1. GitHubの [Code Adventure メインページ](https://github.com/r-carroll/code-adventure) に移動します。
2. ファイルリストの右上にある緑色の **"<> Code"** ボタンを探してください。
3. ボタンを押して、 **"Download ZIP"** を選びます。
4. ダウンロードが終わったら、そのファイル（大体「ダウンロード」フォルダにあります）を右クリックして、 **「すべて展開」** または **「解凍」** してください。
5. 展開してできたフォルダを **デスクトップ** に移動しておくと、後で見つけやすくなります。

> **プロのアドバイス:** すでにGitの使い方を知っている人は、ターミナルで `git clone https://github.com/r-carroll/code-adventure.git` と入力してもOKです！

### 1. Rubyをインストールする
Rubyは、コンピューターに命令を伝えるための言葉です。
- **Windows:** [RubyInstaller.org](https://rubyinstaller.org/) に行き、**Ruby+Devkit 3.3.x (x64)** をダウンロードしてインストールしてください。設定はすべて「次へ」で大丈夫です！
- **Mac:** 「ターミナル」を開いて `brew install ruby` と入力します（Homebrewがある場合）。

### 2. VS Codeをインストールする
ここは、コードを書いたり見たりする場所です。
- ここからダウンロード：[code.visualstudio.com](https://code.visualstudio.com/)
- **重要:** インストールしたら、VS Codeを開いて `Ctrl+Shift+P` を押し、"shell command" と入力して、 **"Shell Command: Install 'code' command in PATH"** をクリックしてください。これで、ターミナルで `code` と打つだけでファイルを開けるようになります！

### 3. Gemini CLIをインストールする
君のAIパートナーです。コードを書くのを手伝ってくれたり、やり方を教えてくれたりします。
- ここからダウンロードしてインストール：[geminicli.com](https://geminicli.com)
- **重要:** 毎回、プログラミングを始める前に、AIに「良い先生」になってもらうための「教育」をしましょう。[SYSTEM_INSTRUCTIONS.txt](./SYSTEM_INSTRUCTIONS.txt) の中身をコピーして、Geminiへの最初のメッセージとして貼り付けてください！

### 4. 「ターミナル」の開き方
ターミナルは、文字だけでコンピューターに命令を出すウィンドウです。
1. キーボードの **Windowsキー** を押します。
2. `cmd` または `PowerShell` と入力して **Enter** を押します。
3. 黒い（または青い）ウィンドウが出てきます。これがターミナルです！
4. プロジェクトのフォルダに移動するには、`cd `（スペースも忘れずに！）と入力してから、`code-adventure` フォルダをそのウィンドウにドラッグ＆ドロップして **Enter** を押してください。

## 🗺 君のミッション：テキストアドベンチャー
ただゲームを作るだけではありません。世界を「作る」方法を学びます。好きなテーマを選んでね：
- 🗡 **侍クエスト:** 古代日本で失われた刀を探す浪人の物語。
- 🤖 **ガンダムパイロット:** 君は宇宙コロニーの最後の希望だ。
- 🧙 **魔法使いの塔:** 論理と呪文を使って魔法の牢獄から脱出しよう。

## 🎓 学び方のコツ（黄金のルール）
AIに「ゲームを作って」とだけ頼むのはやめましょう。それだと、AIは賢くなりますが、**君は賢くなれません。**

代わりに、こんな**パワー・プロンプト（魔法の呪文）**を使ってみよう：
1. *"プレイヤーが武器を選ぶゲームを作りたいです。Rubyでどう書くか教えて。あと、`gets` という命令が何をしているか説明して。"*
2. *"なんでここで `if` を使ったの？ 変えたらどうなる？"*

## 💡 困ったときは（トラブルシューティング）
もし行き詰まったり、ターミナルに怖いエラーメッセージが出たりしても大丈夫：
1. **落ち着いて！** エラーはコンピューターが「困っている」というサインです。
2. **エラーをコピーする:** エラーメッセージ全体をコピーしてください（大体 `Traceback` や `error` という言葉が入っています）。
3. **Geminiに聞く:** エラーを貼り付けて、こう聞いてみよう： *"Rubyのコードでこのエラーが出ました。どういう意味か、どうやって直せばいいか教えて。"*
