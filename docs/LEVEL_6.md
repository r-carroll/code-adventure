# Level 6: Mapping the World (Hashes) 🗺️

Arrays are great for lists, but **Hashes** are great for descriptions. A Hash pairs a "Key" with a "Value".

### Your Task
Ask Gemini:
> "I want to create a map for my game. Can you show me how to use a `Hash` in Ruby where the 'Key' is the room name (like 'Forest') and the 'Value' is the description? Show me how to look up a description using the room name."

### What to look for:
- `world_map = { "Forest" => "A dark woods..." }`: This is how you create a Hash.
- `world_map["Forest"]`: This is how you "get" the value out.

### Deep Dive:
- **Key-Value Pairs:** Think of a Hash like a real dictionary. The word you look up is the **Key**, and the definition is the **Value**.

### Think about this:
- Could you use a Hash to store a monster's stats? Like `{ "name" => "Goblin", "hp" => 50 }`? Ask Gemini to show you how!
