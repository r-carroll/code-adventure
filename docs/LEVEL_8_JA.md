# レベル8：ダイスを振れ（乱数） 🎲

毎回同じことしか起きないゲームはつまらないですよね。 `rand`（ランド）を使って、「ランダム」な要素を加えましょう！

### タスク
Geminiにこう聞いてみよう：
> "戦闘をランダムにしたいです。 `rand(1..20)` を使って、20面ダイスを振るシミュレーションをする方法を教えて。それを使って、攻撃が『命中』したか『ミス』したか決めるにはどうすればいい？"

### 注目ポイント：
- `rand(1..10)`: 1から10の間のランダムな数字を出します。
- `if dice_roll > 15`: これで「難易度チェック」が作れます。

### もっと詳しく：
- **疑似乱数（ぎじらんすう）:** 実はコンピューターは本当のランダムは苦手です！ 複雑な計算でランダムっぽく「見せかけて」います。どうやってランダムな数を作っているのか、Geminiに聞いてみてね。

### 考えてみよう：
- もしダイスで「20」がぴったり出たら「クリティカルヒット（会心の一撃）」にするにはどうすればいいかな？
- プレイヤーに「運（Luck）」のステータスを作って、ダイスの結果にプラスするようにしてみよう！
