# Level 8: The Roll of the Dice (Randomness) 🎲

A game is boring if the same thing happens every time. We use `rand` to add randomness!

### Your Task
Ask Gemini:
> "I want my combat to be random. Can you show me how to use `rand(1..20)` to simulate a 20-sided dice roll? How can I use that to decide if an attack hits or misses?"

### What to look for:
- `rand(1..10)`: Gives you a random number between 1 and 10.
- `if dice_roll > 15`: This creates a "difficulty check."

### Deep Dive:
- **Pseudo-randomness:** Computers aren't actually random! They use complex math to "fake" randomness. Ask Gemini how `rand` really works.

### Think about this:
- How could you make it so that a "Critical Hit" happens if the player rolls exactly a 20? 
- Try to make a "Luck" stat that adds a bonus to your roll!
