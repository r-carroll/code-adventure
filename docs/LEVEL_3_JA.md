# レベル3：終わらない冒険（ループ） 🔄

プレイヤーが勝つか負けるまで、ゲームを続けたいときはどうすればいいでしょう？ そのためには `loop`（ループ）を使います。

### タスク
Geminiにこう聞いてみよう：
> "ゲームに '体力（HP）' を加えたいです。プレイヤーが '探索する' を選ぶたびにHPが10減るようなループを作れるかな？ プレイヤーが 'やめる' と入力した時、または HPが0になった時にループが終わるようにしたいです。`||` (または) や `&&` (かつ) を使って、2つの条件を組み合わせる方法を教えて。"

### 注目ポイント：
- `while` や `loop`: コードを繰り返すようにコンピューターに命令します。
- `break`: ループから抜け出すための「非常出口」です。
- `&&` (かつ) / `||` (または): ルールを組み合わせる時に使います。

### もっと詳しく：
- **真偽値（しんぎち / Booleans）:** `hp > 0` のような条件は、正しい（True）か正しくない（False）かのどちらかになります。
- **論理演算（ろんりえんざん）:** `&&` や `||` を使うと、「鍵を持っていて、かつドアの前にいるなら…」といった複雑なルールが作れます。

### 考えてみよう：
- 10まで数を数えるループを作れるかな？
- もし `break` を書き忘れたらどうなるだろう？（もしプログラムが止まらなくなったら、ターミナルで `Ctrl + C` を押せば強制終了できるから安心 seat!）
