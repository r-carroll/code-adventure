# レベル1：魔法のやまびこ 🗣️

プログラミングで最初にやることは、コンピューターにしゃべらせることです。Rubyでは `puts` という命令を使います。

### 最初のタスク
1. このフォルダでターミナルを開きます。
2. `code adventure.rb` と入力して、VS Codeで新しいファイルを開きます。
3. ターミナルで `gemini` と入力し、その後にダブルクォーテーション `"` で囲った質問を入力します：
   > `gemini "Rubyを勉強しています。僕の名前と年齢を聞いてくれるコードを書いてください。それから、僕が100歳になるのは西暦何年かを計算して教えてくれるようにしたいです。文字を数字に変える方法も説明して！"`

### 注目ポイント：
- `puts`: 画面に文字を表示します。
- `gets.chomp`: 君が文字を入力するのを待ちます。`chomp` は、入力した時の「Enterキー」の余計なデータを取り除いてくれます。
- `#{name}`: 文の中に変数をはめ込むための魔法の枠です。

### もっと詳しく：
- **変数（へんすう）:** `name` や `age` は、コンピューターが情報を入れておくための「小さな箱」だと考えてください。
- **文字列（String）と整数（Integer）:** コンピューターにとって "12" はただの文字ですが、計算するためには「数字」として教える必要があります。Rubyでは `.to_i`（to integer = 整数へ）を使って変換します。

### 考えてみよう：
- 文字と文字を足し算したらどうなるかな？（例： `"こんにちは" + "世界"`）
- `.to_i` を使わずに、文字と数字を足そうとしたらどうなるかな？ やってみて、エラーが出たらGeminiに理由を聞いてみよう！
