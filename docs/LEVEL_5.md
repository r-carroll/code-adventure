# Level 5: The Hero's Backpack (Arrays) 🎒

In a game, you need to keep track of many things, like items in your inventory. For this, we use an **Array** (a list).

### Your Task
Ask Gemini:
> "I want to add an inventory to my game. Can you show me how to create an `Array` in Ruby to hold items like 'Sword' and 'Potion'? Also, show me how to `push` a new item into the list and how to check how many items are in the backpack."

### What to look for:
- `inventory = []`: This creates an empty list.
- `<<` or `.push`: These add things to the end of the list.
- `inventory.include?("Sword")`: This checks if the sword is in the bag.

### Deep Dive:
- **Index:** Computers start counting at **0**. So the first item in your array is at position 0, the second is at position 1. Ask Gemini why programmers start counting at zero!
- **Methods on Arrays:** Lists have "built-in" skills. You can `.sort` them, `.reverse` them, or even `.shuffle` them.

### Think about this:
- How would you remove an item if the player uses a potion? Ask Gemini about the `.delete` method!
