# レベル6：世界の地図（ハッシュ） 🗺️

配列（リスト）は順番に並べるのが得意ですが、**Hash**（ハッシュ）は「説明」を付けるのが得意です。ハッシュは「キー（鍵）」と「値（中身）」をペアにします。

### タスク
Geminiにこう聞いてみよう：
> "ゲームのマップを作りたいです。Rubyの `Hash` を使って、'キー' を部屋の名前（例：'森'）、'値' をその説明にする方法を教えて。部屋の名前を使って、説明を取り出す方法も教えて。"

### 注目ポイント：
- `world_map = { "森" => "暗い森が広がっている…" }`: ハッシュの作り方です。
- `world_map["森"]`: 値を「取り出す」方法です。

### もっと詳しく：
- **キーと値のペア:** ハッシュは本物の「辞書」のようなものだと考えてください。調べたい言葉が **Key** で、その意味が **Value** です。
