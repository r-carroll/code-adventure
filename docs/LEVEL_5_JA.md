# レベル5：勇者のバックパック（配列） 🎒

ゲームでは、持ち物（インベントリ）のようにたくさんの情報を管理する必要があります。そのために、**Array**（配列、つまり「リスト」）を使います。

### タスク
Geminiにこう聞いてみよう：
> "ゲームに持ち物リストを作りたいです。Rubyで '剣' や 'ポーション' を入れるための `Array` の作り方を教えて。あと、新しいアイテムをリストに `push`（追加）する方法と、バックパックに何個アイテムが入っているか数える方法も教えて。"

### 注目ポイント：
- `inventory = []`: 空（から）のリストを作ります。
- `<<` または `.push`: リストの最後に中身を追加します。
- `inventory.include?("剣")`: リストの中に「剣」があるかどうかを確認します。

### もっと詳しく：
- **添字（そえじ / Index）:** コンピューターは数を **0** から数え始めます。リストの1番目は0番、2番目は1番になります。なぜ0から始まるのか、Geminiに聞いてみてね！
- **配列のメソッド:** リストには便利な「組み込みのスキル」があります。並べ替え（ `.sort` ）、逆順（ `.reverse` ）、シャッフル（ `.shuffle` ）などができます。
