# レベル9：物語を保存する（ファイル操作） 💾

プレイヤーが途中で休憩して、あとで続きから遊びたいと思ったらどうしましょう？ 進み具合を「ファイル」に保存（セーブ）する必要があります。

### タスク
Geminiにこう聞いてみよう：
> "プレイヤーの名前とHPを `save.txt` という名前のテキストファイルに保存する方法を教えて。それから、ゲームが始まる時にそのファイルを読み込んで、続きから遊べるようにする方法も教えて。"

### 注目ポイント：
- `File.write("save.txt", data)`: ファイルを作って、そこにデータを書き込みます。
- `File.read("save.txt")`: ファイルからデータを読み取ります。

### もっと詳しく：
- **永続性（えいぞくせい / Persistence）:** 難しい言葉ですが、「電源を切ってもデータが残ること」を意味します。ファイルは、永続性を持たせるための一番基本的な方法です。

### 考えてみよう：
- もし `save.txt` がまだ作られていなかったらどうなるかな？ ゲームが止まらないように `File.exist?` を使って確認する方法をGeminiに聞いてみよう！
