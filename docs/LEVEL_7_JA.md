# レベル7：シンボルと秘密（シンボル） 💎

Rubyでは、ハッシュのキーなどに文字列の代わりに **Symbol**（シンボル）をよく使います。 `:forest` のように書きます。

### タスク
Geminiにこう聞いてみよう：
> "文字列の `"forest"` と、シンボルの `:forest` は何が違うの？ マップのハッシュをシンボルを使って書き換える方法を教えて。なぜシンボルのほうがコンピューターにとって速いのかも説明して。"

### 注目ポイント：
- `:name`: 最初にコロンがついているのがシンボルです。
- `status = :playing`: ゲームの「状態（State）」（プレイヤーが `:alive`（生存）、`:dead`（死亡）、`:winner`（勝利）のどれか）を記録するのに最適です。

### もっと詳しく：
- **イミュータブル（不変）:** シンボルは一度作ったら変えることができません。だから、名前やラベルとして使うときにとても信頼できるんです。

### 考えてみよう：
- `current_state` という変数を使って、ゲームを続けるかどうか決めるにはどうすればいいかな？
- `while current_state == :playing` という書き方について、Geminiに詳しく聞いてみよう！
